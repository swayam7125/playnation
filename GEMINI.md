
# Project Overview

This is a React web application built with Vite. It uses Tailwind CSS for styling and Supabase for the backend. The application is a platform for booking sports venues.

# Building and Running

## Development

To run the application in development mode, use the following command:

```bash
npm run dev
```

## Building for Production

To build the application for production, use the following command:

```bash
npm run build
```

## Testing

To run the tests, use the following command:

```bash
npm run test
```

# Development Conventions

## Styling

The project uses Tailwind CSS for styling. Please adhere to the Tailwind CSS utility-first methodology.

## Backend

The project uses Supabase for the backend. The Supabase configuration is located in `supabase/config.toml`.

## Linting

The project uses ESLint for linting. Please ensure that your code adheres to the ESLint rules. You can run the linter with the following command:

```bash
npm run lint
```
